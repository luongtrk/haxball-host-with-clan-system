# ⚽ Haxball Host With Clan System

![status](https://img.shields.io/badge/status-active-green)
## ⚙️ System Overview

This project is a heavily customized Haxball room server built on top of an original base (inspired by shelld3v), redesigned into a full-featured competitive system with ranking, clan-style matches, anti-toxic moderation, and dynamic match events.

## 🎮 Core Game Modes
- Pick Mode (captain-based selection system)
- Random Mode (auto balance teams)

## 🧠 Match System
- AFK detection
- Substitution system
- Time limit & score limit control
- Auto restart between matches

## 🏆 Player Rating System
- Goals, assists, passes
- Defensive actions
- Mistakes tracking
- Performance-based scoring

## 🛡️ Admin System
- Admin / Super Admin authentication
- Kick / ban / control match system

## 🚨 Anti Toxic System
- Bad word filtering
- Warning system
- Auto punish (kick/ban)

## 🗳️ VoteKick System
- Player votekick
- Anti abuse limit system

## 🎨 Visual & Match System
- Dynamic team colors
- Special match detection (derbies, events)
- Goalkeeper color system

## 📊 Commentary System
- Live goal commentary
- Score-based messages
- Penalty feedback system

## 📡 Discord Integration
- Match logs
- Stats tracking
- Reports system

## 🏟️ Stadiums
- Training map
- 1v1 / 2v2 maps
